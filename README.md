**Black Box Testing – Kara Kutu Testi Dokümantasyonu**

Bu proje, Yazılım Testi dersi kapsamında hazırlanan Kara Kutu Testi (Black-Box Testing) tekniklerini içeren dokümantasyon çalışmasıdır.
Dokümanda aşağıdaki test teknikleri açıklanmış ve örneklerle gösterilmiştir:

 - Eş Değerlere Bölme (Equivalence Partitioning)
 - Uç Değer Analizi (Boundary Value Analysis)
 - Karar Tablosu Testi (Decision Table Testing)

Ek olarak, her teknik için senaryolar, giriş verileri ve beklenen çıktılar örneklenmiştir.
